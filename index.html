Services

We provide different services to uplift refugee girls with high potential to be ableto contibute in solving problems in their respectives commnunity but also to the role models in their community. The courses that we provide take student from basic to advanced level and become a professional in areas like Digital Skills, Software Engineering as well as Embedded Robotics Engineer

Digital Skills

In this program, students with no experience in the computer will be introduced to the basics of computers. Understanding how the computer works from hardware to software, basic typing skills, understand how to use Microsoft 365 packages (Microsoft word processing, Microsoft Excel as well as Microsoft PowerPoint presentation), to internet. However, this program is also much focused on how to become a power user of Microsoft Excel so that student students who are aspired to start their professional carrier in Excel can directly jump in and start looking for job opportunities and be able to work as professional Excel users of for business.

Web development

Because most companies and organizations are looking forward to going into the digital transformation of the way they process data, then there is a need to have professionals and developers to help them transform their needs into reality. This course is designed to help students who are aspiring in starting their professional carrier in web development. Students in this course are introduced to the Front-end and backend web technologies that are most in-demand in many industries.

Program Duration Key Focus
Digital Literacy 2 Months Introduction to basic computer, Micorsoft 365(Microsoft word processing, Microsoft Excel as well as Microsoft PowerPoint presentation) and introduction to internet

Frontend Web Development

6 Months

The frontend web development course covers the introduction to HTML, CSS, Bootstrap, JavaScript, React JS as well as Git and GitHub. These contents are coved to help students to be able to start their carrier as front-end web developers because this is not a very theoretical course rather it is a very hands-on course where students learn by developing real-world projects to help them solve problems and create solutions that the community will as well benefits.

These contents are coved to help students to be able to start their carrier as either frontend or backend web developers.

Backend Web Development 6 Months

Because every front-end technology needs to have a way of storing and manipulating the data that they provide, then students who have taken the front-end course can either choose to take this course to become Full Stack developers. This course teaches how to store, manage and manipulate data using Database technologies by learning the Structured Query Language, Python, Django as well as Git, and GitHub.

Embedded Robotics

Students in this class are introduced to basic electronics and electronic devices used in developing embedded systems (devices like resistors, capacitors, and inductors); these basics even go further into the active elements like diodes and transistors and how to build and design circuits and systems on the development board. After completing these introductions students also look at how to use different Microcontroller devices like Arduino and Raspberry Pi to build, monitor, and control systems. The students then learn how to control the environment and react to the environment using Sensors and Actuators.

The course covers a variety of ranges of sensors including Ultrasonic sensors, Light Controlled sensors, MQ5 Gas sensors, soil moisture sensors, Optical sensors, Electro-Chemical Sensors, Motion sensors, and other types of sensors. Students learn to also build systems that are intelligent by applying computer vision techniques in order to be able to react to the external environment intelligently.
